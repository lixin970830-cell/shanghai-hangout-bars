# Shanghai Hangout Bars

适合聊天、约会、续摊和下班小喝的上海酒吧精选清单。

Live site:

https://lixin970830-cell.github.io/shanghai-hangout-bars/

## Local Preview

```bash
npm install
npm run dev
```

Static preview:

```bash
python3 -m http.server 5175
```

Then open:

```txt
http://127.0.0.1:5175/static.html
```

## Public Website

The GitHub Pages version is in:

```txt
docs/index.html
```

After pushing to GitHub, enable GitHub Pages:

```txt
Settings -> Pages -> Deploy from a branch -> main / docs
```

## Stack

- React
- Vite
- CSS
- LocalStorage
